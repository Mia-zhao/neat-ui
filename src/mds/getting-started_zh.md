# 开始使用

请先[安装](#/doc/install) Neat UI 组件库。

然后通过以下代码引用组件:
```
import { Button, Switch, Tabs, Menu, Modal, Carousel } from 'neat-ui'
```
接下来就可以开始使用组件了。

## Vue 组件使用示例
```
<template>
  <Button>Hello</Button>
</template>

<script lang="ts">
import { Button } from 'neat-ui'
export default {
  components: {
    Button
  }
}
</script>
```
请参考组件列表获得更多示例。
